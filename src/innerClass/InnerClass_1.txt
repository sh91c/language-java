[내부 클래스]
- 클래스 안의 클래스

class A {               class B {
    ...                     ...
}                       }

- 내부 클래스를 사용한다면..
class A {
    ...
    class B {
        ...     // 객체생성 없이 A의 멤버에 접근 가능하다.
    }
}

[내부 클래스 장점]
- 내부 클래스에서 외부 클래스의 멤버들을 쉽게 접근할 수 있다.
- 코드의 복잡성을 줄일 수 있다. (캡슐화)
    ㄴ 위 코드에서 클래스 B가 클래스 A 내부에서만 사용할 것이라고 가정,
       타 클래스에서 클래스 B를 쉽게 접근하지 못함(가능은 하다.)
